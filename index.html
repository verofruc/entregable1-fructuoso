Carrito de compras

Este código crea arrito de compras. El HTML define la estructura básica de la página, que incluye una lista de productos que se pueden agregar al carrito y una sección que muestra los ítems añadidos al carrito, así como el total de la compra. Los botones permiten agregar productos al carrito, borrar todos los ítems del carrito, o proceder al pago.

El código JavaScript se encarga de manejar la lógica de la aplicación. Utiliza la función 'agregarAlCarrito' para solicitar al usuario la cantidad de productos que desea añadir al carrito mediante un prompt. Luego, actualiza el carrito y el total mostrado en la página con la función 'actualizarCarrito'. La función 'borrarTodosLosItems' permite eliminar todos los ítems del carrito después de confirmar la acción con un confirm. Finalmente, la función pagar muestra un mensaje de agradecimiento y limpia el carrito si se confirma el pago. Si el carrito está vacío, se muestra un mensaje de alerta.
